# :art: framework team gallery

## Запуск проекта

Склонировать репозиторий

```bash
$ git clone https://github.com/str0yka/just-a-blog.git
```

Установить зависимости

```bash
$ yarn
```

Запустить проект

```bash
$ yarn dev
```
